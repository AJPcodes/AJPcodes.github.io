--- layout: post title: "Manually installing Mongo Hacker for windows" date: 2016-02-08 22:00:00 -0600 blurb: How to get Mongo Hacker in the Mongo Shell for windows ---

So I was watching a team treehouse introductory MongoDB tutorial, and wanted to follow along with some of the mongo-hacker added commands. If I were in a Mac environment, that would have been as simple as an NPM install. I'm working in Windows, though so it turned into a it more of a project.

Luckily. I found a great article on Stack Overflow The solution ended up being to grab a zip file of the Mongo Hacker files, concatenating all of the Mongo Hacker 'Hacks' js files, and pasting them below the config.js files into a .mongorc.js file. I'd be lying if I said I didn't have fun.

Happy coding
